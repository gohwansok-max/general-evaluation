✨ AI 새 문제 만들기 (선택)

Google Gemini API 키를 넣으면 매번 완전히 새로운 문제를 무제한으로 만들어 줍니다. 키는 이 기기에만 저장되고 외부로 나가지 않아요. 키가 없어도 ‘새 문제’ 버튼은 기본 문제은행에서 매번 다르게 출제됩니다.

📱 폰에 설치 — 폰에서 이 파일을 열고 · 아이폰: 공유→‘홈 화면에 추가’ · 안드로이드: 메뉴(⋮)→‘홈 화면에 추가’
🎧 듣기는 🔊 버튼을 누르면 기기 내장 음성으로 영어를 읽어 줍니다.
